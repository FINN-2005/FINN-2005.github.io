About SimLab

SimLab is a collection of interactive simulations designed to explain core concepts used in game development.

Many systems in game engines rely heavily on mathematics and physics. Concepts like vector math, simulation loops, and physical forces form the foundation of rendering, movement, and interaction in games.

This site aims to break down these ideas using visual and interactive examples. Each article includes a simulation built with JavaScript and p5.js so the concepts can be explored directly.

Topics Covered

  • Vector math and dot products
  • Spring physics and oscillation
  • The classic game loop architecture

Technology

  • FastAPI for routing
  • Jinja templating
  • JavaScript + p5.js for simulations
